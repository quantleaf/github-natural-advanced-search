Natural Advanced Search for GitHub

Usage guide

Perform search

When 'Advanced search' result alternativ appear, click on it to perform it or select it by using they arrow keys.

Suggestions

Suggestions are helping you do searches, they are based on what you can continue to type. You can press (ctrl) + (space) to show/hide more suggestions.

If the interpretation is wrong

Try to be more precise. For example, you can change

invert binary tree and creation date from one year ago

to

invert binary tree and date > 2015-01-01

In some cases, text properties might be interpreted incorrectly. You can prevent this by applying quations, " ... " around the words.

content "invert binary tree"

Dates

You can search by dates in multiple ways:

creation date from 2005-05-05 to 2006-06-06

creation date from three years ago

creation date from one year ago

You can query all date fields that exist in the advanced search

Natural Advanced Search for GitHub

Usage guide

Perform search

When 'Advanced search' result alternativ appear, click on it to perform it or select it by using the arrow keys.

Suggestions

Suggestions are helping you do searches, they are based on what you can continue to type. You can press (ctrl) + (space) to show/hide more suggestions.

Combining conditions

Use 'and' or '&&' to combine multiple conditions. Most of the time this might not be needed as the tool automatically detects when multiple conditions are present.

NOT condition

You can make 'not' conditions for all properties. For example:

not owner = USERNAME

owner != USERNAME

owner not equals to USERNAME

If the interpretation is wrong

Try to be more precise. For example, you can change

content invert binary tree and creation date from one year ago

to

content = invert binary tree && date > 2015-01-01

In some cases, text properties might be interpreted incorrectly. You can prevent this by inserting quotations around the words:

content "invert binary tree"

Dates

You can search by dates in multiple ways, some examples:

creation date from 2005-05-05 to 2006-06-06

creation date from one year ago

creation date before one year ago

A list of all searchable properties

Read the official documentation for the advanced search. If you find a property you want to query, just write it but replace the colon with an equality symbol, e.g. 'owner:USERNAME' becomes 'owner = USERNAME' or 'owner equals USERNAME'.
